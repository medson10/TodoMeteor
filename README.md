# Todo Meteor
A Todo application built with Meteor, React and MongoDB

## Prerequisites

Node

Meteor

MongoDB

## Clone the repository
```
git clone https://github.com/medson10/TodoMeteor.git
```
## Open the directory
```
cd TodoMeteor
```

## Install dependencies run the application
```
meteor npm install
```

## Run the application
```
meteor
```

## Access the application in
```
http:localhost:3000/
```
